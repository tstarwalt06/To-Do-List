# To-Do List App

## Description
Simple to-do list web app built with HTML, CSS, and JavaScript.  
Allows users to create, manage, and filter tasks with tags.  
All data is saved using localStorage, and tasks remain after refreshing the page.

## Features

### Core Features
- Add a task with a required name
- Optional tag (school, work, personal, etc.)
- Display tasks in a list
- Mark tasks as complete
- Delete tasks

### Filters
- Filter by:
  - All tasks
  - Active tasks
  - Completed tasks
- Filter by tag using a dropdown

### Persistence
- Tasks are saved in localStorage
- Tasks remain after page reload

### Validation and UX
- Error message if task name is blank
- Form clears after adding a task
- Displays active task counter

## Technologies Used
- HTML
- CSS
- JavaScript
- localStorage
